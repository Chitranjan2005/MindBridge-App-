## 🔧 File Purposes

### Root Configuration Files

- **package.json** - Root scripts for running both frontend & backend
- **.gitignore** - Files to exclude from git
- **setup.bat** - Windows setup automation
- **setup.sh** - Mac/Linux setup automation

### Backend Files

- **server.js** - Express server entry point
- **chatController.js** - Handles /api/chat and /api/analyze
- **chatRoutes.js** - Defines API routes
- **aiService.js** - Claude/OpenAI integration
- **emotionDetection.js** - Keyword-based emotion analysis
- **.env.example** - Environment variables template

### Frontend Files

- **index.html** - HTML entry point
- **main.jsx** - React entry point
- **App.jsx** - Root React component
- **ChatPage.jsx** - Main chat interface
- **MessageBubble.jsx** - Individual message display
- **InputBox.jsx** - Text input component
- **TypingIndicator.jsx** - Loading animation
- **index.css** - Global styles
- **vite.config.js** - Vite configuration
- **tailwind.config.js** - Tailwind CSS config
